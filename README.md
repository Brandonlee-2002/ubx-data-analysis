# UBX World Project

I joined UBX World as a data science intern. Together with Dan Blike, my former boss, we collaborated and brainstormed ideas for exploratory research towards a new badminton rating system. We used R programming initially due to me using it in class for more practice with RStudio, but realized that RStudio could not handle to datasets that I was trying to open. I then switched over to Python because Python is generally more used in the workforce and would be a great skill to polish and work on to prepare for future opportunities. I initially parsed through large datasets and sucessfully created a graph algorithm that allows us to compare players from thousands of regions and tournaments.
